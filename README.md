# Viaggiorni

A desktop journal app built for daily use. Local-first, fully encrypted, no accounts, no cloud.

Write entries, track habits, log mood, manage todos — all stored in an encrypted SQLite database on your machine. An optional Ollama integration adds a writing prompt and word of the day if you have it running locally.

Built with Electron, React, TypeScript, and SQLite.
